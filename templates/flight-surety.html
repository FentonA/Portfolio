003 :Flight Surety

This project is a flight insurance Dapp

It allows you to purhcase insurances for flight delays for passengers with ether

It is meant to emulate a managed collaboration between multiple airlines

Features a security mechanism within the smart contracts that only allows other registered airlines to vet and register a new airline into the group

Should a flight be delayed due to airline fault, passengers are reimburesed 1.5x the amount that they paid the insurance for in ether

Tech Stack

Project Tech Stack includes:

  • Blockchain orales for data pool
  • webpack and babel for front end server deployment
  • Javascript, web3.js and truffle.js used to connect project to ethereum network and test smart contract
  • Solidity used for smart contract development

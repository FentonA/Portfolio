002 : Decentralized Supply Chain

This project provides a supply chain solution integrated with the Ethereum network

It features smart contracts that provides the ability to track user specific permission controls throughout the supply chain

It also includes smart contracts that helps track and verify a products authenticity.

Tech Stack

This project's Tech Stack includes:

  • Ethereum Rinkeby Network
  • Solidity used to develop smart contracts
  • Node.js, web3, Truffle.js amd metamask used to connect to ethereum network and test smart contracts
  • HTML CSS and Javscript for project frontend

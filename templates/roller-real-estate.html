001 :Decentralized Real Estate Marketplace

This project leverages the functionanlity of ethereum smart contracts to create an online real-estate market place

Each of the items in the marketplace are created with Ethereum's ERC721 tokens, which turns the houses and apartments into NFTs, providing them with

with unique identities that can't be replicated

The project also features an ERC20 token dubbed 'Roller Tokens' which are used to buy and sell in the various property

Lastly, all the metadata needed to create the list of items are stored in aws hosting, however IPFS can easily be integrated to replace AWS

Tech Stack

Project Tech Stack includes:

  • Rinkeby Ethereym Network
  • AWS used the hosting for the real estate metadata
  • Javascript, web3.js and truffle.js used to connect project to ethereum network and test smart contract
  • list of real estate ERC721 tokens listed on opensea, with custom ERc20 token buiilt for transactions
  • Solidity used for smart contract development
